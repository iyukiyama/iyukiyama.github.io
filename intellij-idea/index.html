IntellJ Idea使用指南

快捷键


将原有工程转为Maven工程


修改工程名

以一个当前工程名为 thread_practice 的工程为例，将工程名改为 concurrency_in_action 。

  1. 首先在 idea 内修改工程名，修改之后工程名后 thread_practice 后会有 [concurrency_in_action] 的字样。

  1. 接着在 Project Structure 设置页按如下修改工程名。

  1. 彻底关闭 idea 。
  2. 在 idea 工程文件夹下将 thread_practice 文件夹名改为 concurrency_in_action 。
  3. 重启 idea，这时会看到原来的 thread_practice 已经变灰，点击后面的 X 删除。
  4. 点击 Open 打开改名后的 concurrency_in_action 文件夹。选择 Trust Project。

  1. 打开后可以看到已经修改成功。

  1. 如果是 Maven 工程，右下角会提示是否要载入 Maven 工程，如下，点击 Load Maven Project 载入。如果没有提示，也可以右键单击 pom.xml 文件，点击 Add as Maven Project 载入。

  1. 如果是 Maven 工程，还要修改 pom.xml 文件中的 artificialId 为新的工程名，并重新载入 Maven 工程 (重新载入所有 Maven 工程 : Reload All Maven Projects；单独载入某个 Maven 工程 : Reload Project) 。
