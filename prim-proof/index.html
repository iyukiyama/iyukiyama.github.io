算法描述

Prim (普里姆算法): 一种基于贪心思想的求解无向图上 MST 的算法。我们直接将 Prim 算法和 Dijkstra 二者对比如下。

我们看到这两个算法的每一步都几乎相同，不同点主要有三处，最主要的是第 3 点。

  1. Prim 用于在「无向图」中求 MST，因此建图时要「双向建边」。Dijkstra 应用于有向图时单向建边，应用于无向图时双向建边。
  2. Dijkstra 中的「距离」指的是顶点到源点的距离。Prim 中的「距离」指的是顶点到其已在生成树上的邻接顶点 (父顶点) 的距离。
  3. 顶点距离更新方式不同 。

算法过程

  1. 建图及初始化。

    1. 构建双向带权图。
    2. 设置一个大小为 ∣V∣|V|∣V∣ 的用于标记顶点距离是否「已确定」的 booleanbooleanboolean 数组 visited[]visited[]visited[] ，下标表示顶点。
    3. 设置一个大小为 ∣V∣|V|∣V∣ 的距离数组 dists[]dists[]dists[] ，下标表示顶点。初始化所有顶点的距离为 InfinityInfinityInfinity ，表示所有顶点的距离尚未确定。
    4. 任意选取 一个顶点作为起始顶点，置距离为 0，通常我们将第一个顶点的距离置为 0 。

  2. 以一个循环寻找 当前距离未确定顶点中距离最小者 uuu ，立即置 uuu 的距离为「已确定」。

  3. 距离更新。尝试更新 uuu 的所有 距离未确定的 邻接顶点 vvv 的距离 dvdvdv。即 dv=min{dv,∣(u,v)∣}dv = min\{dv, |(u,v)|\}dv=min{dv,∣(u,v)∣} 。

  4. 循环结束时，所有顶点距离被求出，也就是 MST 所有边权被求出。


正确性证明

如下证明参考了 参考1 以及 参考2。

虽然 Prim 算法过程与 Dijkstra 类似，但证明过程并不通用。如下是 Prim 算法的证明过程。

证明：在图 GGG 上执行 Prim 算法得到的生成树 SSS 是 MST。

  1. 记 GGG 上的 MST (之一) 是生成树 SminS_{min}Smin​，即证明 ∣S∣=∣Smin∣|S| = |S_{min}|∣S∣=∣Smin​∣ ，绝对值表示生成树的边权和。
  2. 在 SSS 生长过程中，首次 遇到的不属于 SminS_{min}Smin​ 的边为 e=(u,v)e=(u, v)e=(u,v) ，uuu 在此前得到的点集中，vvv 在剩余的顶点中。从 SSS 中拿走 eee ，则 SSS 剩下的不连通的两部分分别为包含 uuu 的子树 UUU 以及包含 vvv 的子树 VVV 。
  3. SminS_{min}Smin​ 是 MST，其上必存在 uuu 到 vvv 的路径，且已知 uuu 在 UUU 中，vvv 在 VVV 中，由于 (u,v)∉Smin(u,v)∉S_{min}(u,v)∈/Smin​ ，则必然存在边 f=(a,b)∈Sminf=(a,b)∈S_{min}f=(a,b)∈Smin​ 穿过 U→VU → VU→V，即 aaa 在 UUU 中， bbb 在 VVV 中 (若 aaa 是 uuu ，则 bbb 不是 vvv；若 bbb 是 vvv，则 aaa 不是 uuu) 。
  4. 当 eee 即将被加入 SSS 时，顶点 aaa 已在 UUU 中 ( UUU 是 SSS 当前生长到的部分)。
    1. 此时 aaa 是「距离未确定」的顶点。因为如果 aaa 是距离已确定的顶点，则在 UUU 中存在一条边 ggg 连接 aaa 和另一顶点，我们已经知道 eee 是 SSS 生长过程中 首次 遇到的不属于 SminS_{min}Smin​ 的边，也就是说 g∈Sming∈S_{min}g∈Smin​ ，这与 f∈Sminf∈S_{min}f∈Smin​ 矛盾，因为 aaa 一旦「距离确定」，后续就不会考察 aaa 的连边，fff 也就不可能加入 SminS_{min}Smin​ 。
    2. 由于 aaa 是「距离未确定」的顶点，考察顶点 uuu 的连边时，顶点 aaa 的连边也一定会被考察。由于 eee 在此轮考察中被 Prim 算法选入 SSS 中，而 fff 未被选择，可知 ∣e∣≤∣f∣|e|≤|f|∣e∣≤∣f∣ 。
  5. 对于 SminS_{min}Smin​ ，断开 f=(a,b)f=(a,b)f=(a,b) ，则 SminS_{min}Smin​ 被分成不连通的两棵子树 AAA 和 BBB，且 u→a→b→vu → a → b → vu→a→b→v 被断开，说明 uuu 与 vvv 分别在不连通的两棵子树 A,BA, BA,B 上。此时连上 e=(u,v)e=(u,v)e=(u,v) 得到的 SAeBS_{AeB}SAeB​ 显然是一棵生成树。因为 ∣e∣≤∣f∣|e|≤|f|∣e∣≤∣f∣ ，则必然有 ∣e∣=∣f∣|e|=|f|∣e∣=∣f∣ ， 否则 SAeBS_{AeB}SAeB​ 边权之和更小，与 SminS_{min}Smin​ 是 MST 矛盾。因此 ∣SAeB∣=∣Smin∣|S_{AeB}|=|S_{min}|∣SAeB​∣=∣Smin​∣，即 SAeBS_{AeB}SAeB​ 也是 MST。
  6. 重复上述过程，继续考察 SSS 中下一条不属于 SminS_{min}Smin​ 的边，直到得到 SSS 。每一次考察，都可以得到上述 5 的结论，最终有 ∣S∣=∣Smin∣|S|=|S_{min}|∣S∣=∣Smin​∣，即 SSS 是 MST。
